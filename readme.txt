## ✅ Step-by-Step Guide to Run the Project

### 1. Requirements

- Java JDK 8 or above
- [NetBeans IDE](https://netbeans.apache.org/) (recommended)
- Internet (for email simulation if needed)

---

### 2. How to Open the Project in NetBeans

1. Open **NetBeans IDE**.
2. Go to **File > Open Project**.
3. Browse to the location of the `Assignment3` folder.
4. Click on the project and then click **Open Project**.

---

### 3. Run the Program

1. In the Projects pane, open `Assignment3 > assignment3 > Assignment3.java`.
2. Right-click `Assignment3.java`.
3. Click **Run File** or press `Shift + F6`.
4. A console will open at the bottom where you can interact with the system through a text menu.

---

### 4. What You Can Do in the Program

- Add new doctors and patients
- Book and manage appointments
- Trigger emergency alerts
- Send notifications via email and SMS
- Simulate chat and video calls using interfaces
- View patient medical history and vitals
